"Deskripsi Tugas yang dilaksanakan"

- Membuat Tampilan Form Login Web Tahoma

1. Persiapkan bahan yang akan digunakan (Gambar Logo, Font, Rancangan Desain).

2. Buka aplikasi Figma buat Frame sesuai Ukuran dan siapkan Rancangan yang sudah dibuat.

3. Kemudian masukkan Gambar Logo dan Font yang sudah dipilih.

4. Tambahkan desain svg untuk memperindah tampilan.

5. Langkah selanjutnya adalah menambahkan form & tombol-tombol untuk melengkapi tampilan, dan desain sudah siap.

6. Lalu buka Software Visual Studio Code Code.

7. Buat Folder, lalu di dalamnya buat File baru bernama login.html (Untuk Halaman Website ) dan style-login.css (Untuk memberi Style pada Halaman Website), lalu buat juga Folder bernama img (Untuk menyimpan semua Asset yang akan digunakan).

8. Kemudian Masukkan semua Asset yang akan digunakan.

9. Buatlah Struktur html sederhana dan juga menghubungkan file css serta ikon dengan menggunakan Library Unicons.

10. Kemudian Ketikkan struktur Code html yang Menyerupai Desain yang sudah dibuat untuk mengisi konten bagian Atas serta mengisi class.

11. Dilanjutkan dengan struktur Code html bagian bawah dan juga mengisi class.

12. Selanjutnya ketikkan Reset serta Font yang diinginkkan ke dalam File css.

13. Kemudian Ketikkan Style pada bagian Container dan atur posisi sesuai desain, Logo serta Judul Website.

14. Ketikkan juga Style pada bagian Header sesuai dengan desain yang sudah dibuat.

15. Dilanjutkan dengan Memberi Style pada bagian Judul dan Logo.

16. Kemudian Pemberian Style pada bagian Form Login.

17. Berikan juga Style pada bagian Button dan Checkbox.

18. Selanjutnya ketikkan Code Style untuk Tombol guest.

19. Dilanjutkan dengan mengetikkan Code Style pada bagian footer.

20. Setelah pemberian Style pada semua element HTML, Selanjutnya mengatur Responsive pada Website.

21. Dilanjutkan dengan Responsive ukuran lebar layar 320px.

22. Responsive ukuran layar 320px pada bagian Form.

23. Responsive ukuran layar 320px pada bagian Tombol guest.

24. Langkah Terakhir yaitu memberikan code Style pada ukuran lebar layar 480px .

25. Setelah itu tampilan Website sudah siap sesuai dengan Desain yang sudah dibuat.
